Enhancing Boost.Intrusive Library

This project is extending the scope of Boost.Intrusive library by adding more advanced data structures.

The data structures implemented are :-

  • Segment tree
  • Fenwick tree
  • Suffix tree
  • Suffix automata

Apart from this 2 more commonly used algorithms are also implemented.They are:-

  • Heavy light decomposition
  • Centroid decomposition

The documentation for all the implemented code is present at Documentation
